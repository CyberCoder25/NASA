NASA

Project Overview
- NASA project is a front-end web application showcasing space data and APIs
- Built to demonstrate advanced JavaScript usage and API integration
- Focused on learning asynchronous requests, DOM manipulation, and interactive UI

Key Highlights
- Dynamic data fetching from NASA APIs
- Interactive and responsive UI
- Fully built with vanilla JavaScript, HTML, and CSS
- Modular and maintainable code structure
